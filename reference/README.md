## PROJECT VISION

### The Core Idea
Political ideologies (communism, socialism, capitalism, anarchy) are not rival teams fighting for total control. They are different **systems operating at different scales** of human relationship—from intimate family to population-level governance. The problem isn't picking the wrong ideology; it's misapplying the right tool to the wrong scale.

### Why This Book Matters
Most political discourse treats these as competing religions. This book shows they're **complementary tools in a shared toolbox**—and that we already use all of them every day without realizing it.

### Target Impact
Reach **I+D+G readers** (Working Geniuses framework) who will:
- Test and verify the frameworks (Discernment)
- Spread verified ideas to others (Galvanizing)
- Implement in their own contexts (readers with Tenacity find their own applications)

**Voice Architecture:** I→D→G (Invention → Discernment → Galvanizing)
- Author provides diagnostic tools (I+D domain)
- Readers verify against their experience (D invitation)
- Readers spread what they've verified (G activation)
- Readers implement in their own T-domain contexts

## TARGET AUDIENCE DEEP DIVE

### Working Geniuses Framework

**Six Geniuses (Project Lifecycle Phases):**
1. **Wonder** - Pondering and speculation
2. **Invention** - Creating and defining ← Joshua's strength
3. **Discernment** - Intuiting and evaluating ← Joshua's strength
4. **Galvanizing** - Rallying and organizing ← TARGET AUDIENCE
5. **Enablement** - Supporting and assisting
6. **Tenacity** - Completing and delivering ← TARGET AUDIENCE

### Why Target G+T (Not I+D)?

**If we write for Invention/Discernment types (like Joshua):**
- ✓ They would appreciate the frameworks
- ✓ They would understand the systems thinking
- ✗ They would NOT rally others
- ✗ They would NOT drive implementation
- ✗ Book goes nowhere

**If we write for Galvanizing + Tenacity types:**
- ✓ They rally others around the ideas (G)
- ✓ They drive projects to completion (T)
- ✓ They create real-world change
- ✓ They multiply the book's impact

**Content Implications:**
- Must inspire action, not just understanding
- Must be rallying-ready (G can use it to organize)
- Must enable completion (T can execute from it)
- Must translate systems → stories they can share
